# Analog Clock using SwiftUI

```Design Pattern:MvvM```

The clockview has the following componnents
1. HourHand (Shape)
2. MinuteHand (Shape)
3. SecondHand (Shape)
4. Line Dial (line used to mark the position of each minute) (Shape)
5. Numeric Dial (hour label) (View)
